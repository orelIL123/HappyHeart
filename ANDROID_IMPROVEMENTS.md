# שיפורים לאנדרואיד

## מה תוקן?

### 1. תיקוני UI באנדרואיד
- **Shadows ו-Elevation**: הוספת תמיכה נכונה ב-shadows באנדרואיד עם `elevation`
- **טקסט**: תיקון בעיות של טקסט שנחתך או בורח מהקונטיינר
- **פונטים**: נעילת font scaling למניעת בעיות תצוגה
- **Safe Area**: padding נוסף באנדרואיד כדי שהתוכן לא יוסתר מהטאב התחתון
- **Buttons**: גובה מינימלי ו-padding נכון לכפתורים

### 2. תיקון קריסה בהרשמה
- הוסרה שימוש לא נכון ב-`register` function מה-AppContext
- כעת משתמשים רק ב-`firebaseService.registerWithEmailAndPassword`

### 3. עדכונים מהירים עם EAS Update
- **בדיקה אוטומטית**: האפליקציה בודקת עדכונים מיד בטעינה
- **Polling אגרסיבי**: בדיקת עדכונים כל 30 שניות
- **עדכון אוטומטי באנדרואיד**: באנדרואיד, העדכון מתבצע אוטומטית ללא אישור המשתמש
- **מהיר ככל האפשר**: fallbackToCacheTimeout=0 לעדכונים מהירים

## איך לשלוח עדכון?

### שלב 1: ערוך את הקוד
```bash
# ערוך את הקבצים שצריך
```

### שלב 2: שלח עדכון ל-EAS
```bash
# לסביבת Preview:
eas update --branch preview --message "תיקוני באגים באנדרואיד"

# לסביבת Production:
eas update --branch production --message "גרסה חדשה עם שיפורי UI"
```

### שלב 3: המתן 30-60 שניות
- המשתמשים יקבלו את העדכון אוטומטית תוך כדקה
- באנדרואיד - העדכון יותקן ללא שאלות
- ב-iOS - יקפוץ אישור

## קבצים שנוצרו/שונו

### קבצים חדשים:
- `constants/AndroidStyles.ts` - פונקציות עזר לסטיילים באנדרואיד

### קבצים ששונו:
- `app/_layout.tsx` - לוגיקת עדכונים מהירה
- `app.json` - הגדרות EAS Update
- `app/(auth)/register.tsx` - תיקון קריסה + UI
- `app/(auth)/login.tsx` - שיפורי UI
- `app/(tabs)/create.tsx` - שיפורי UI ו-padding
- `components/ActivityCard.tsx` - שיפורי UI

## טיפים

1. **עדכון מהיר בפיתוח**: השתמש ב-`eas update` במקום build מלא
2. **בדיקה**: פתח את האפליקציה באנדרואיד וסגור אותה - ב-30 שניות הבאות תתקבל עדכון אם יש
3. **Debug**: הסתכל על ה-console logs - יש הדפסות של בדיקת עדכונים

## בעיות נפוצות

**Q: העדכון לא מגיע?**
A: בדוק ש-`eas update` רץ בהצלחה ושה-branch נכון (preview/production)

**Q: האפליקציה לא עובדת אחרי עדכון?**
A: שלח `eas update` חדש עם תיקון, או עשה rollback עם EAS dashboard

**Q: רוצה לבטל עדכון אוטומטי?**
A: שנה את `app/_layout.tsx` - הסר את ה-interval והחזר את האלרט ב-iOS
